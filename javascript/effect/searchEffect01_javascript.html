indexOf()를 이용하여 검색하기

JAVASCRIPT 속성 검색하기

  • slice() : slice() 문자열에서 원하는 값을 추출 : 반환 (문자열)
  • substring() : substring() 문자열에서 원하는 값을 추출 : 반환
  • substr() : substr() 문자열에서 원하는 값을 추출 : 반환 (문자열)
  • split() :split() 문자열에서 원하는 값을 추출 : 반환 (배열)
  • indexOf() : indexOf() 문자열에서 특정 문자의 위치를 찾음 : 반환(숫자)
  • trim() : trim() 문자열 앞뒤 공백 제거 : 반환 문자열
